# Coevolved Sumobots


## About
This project applies concepts from within the field of coevolution to evolve complex robot behavior within the context of a "Sumo" fighting ring.

## Dependencies
- Python 2.7
- ROS Melodic
- Gazebo
- NEAT
  - ```$ pip install neat-python```
- Graphviz
  - ```$ pip install graphviz```
